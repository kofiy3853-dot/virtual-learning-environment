  WHAT YOU NEED TO DO
================================================================================

STEP 1: Create MongoDB Atlas Account (15 minutes)
   - Go to mongodb.com/cloud/atlas
   - Create free cluster
   - Create database user
   - Get connection string
   - Configure IP whitelist

STEP 2: Create Render Account (5 minutes)
   - Go to render.com
   - Connect GitHub repository

STEP 3: Deploy Backend (20 minutes)
   - Create web service
   - Set environment variables (including MONGO_URI)
   - Deploy

STEP 4: Deploy Frontend (20 minutes)
   - Create web service
   - Set environment variables (NEXT_PUBLIC_API_URL)
   - Deploy

STEP 5: Test (15 minutes)
   - Test registration
   - Test login
   - Test dashboard redirect
   - Verify logs

TOTAL TIME: ~75 minutes (1 hour 15 minutes)

  JWT_SECRET = your-secure-random-string-here
  CLIENT_URL = https://your-frontend.onrender.com

Frontend (Render):
  NEXT_PUBLIC_API_URL = https://your-backend.onrender.com
  NODE_ENV = production

================================================================================
  TROUBLESHOOTING
================================================================================

MongoDB Connection Error?
  → Set MONGO_URI environment variable on Render
  → Check MongoDB Atlas IP whitelist
  → Verify connection string format

401 Unauthorized?
  → Verify JWT_SECRET is set
  → Check user exists in database
  → Verify token is being sent

CORS Errors?
  → Verify CLIENT_URL matches frontend URL
  → Verify NEXT_PUBLIC_API_URL matches backend URL

403 Forbidden?
  → Check user role in database
  → Verify authorization middleware

See DEPLOYMENT_ACTION_ITEMS.md for more troubleshooting.
